# Proyecto de Remodelación Web - Grúas Porras Marbella

**[Ver el proyecto en vivo aquí](https://aarcesim-git.github.io/Modelo_GruasPorrasMarbella/)**


---

## ¿De qué trata este proyecto?

Este proyecto es una propuesta libre de rediseño y modernización para la página web real de **Grúas Porras Marbella** (https://www.gpm2023.com/), una empresa local de asistencia en carretera y transporte de vehículos. 

El objetivo principal de esta remodelación ha sido actualizar la presencia digital del negocio, creando una interfaz mucho más limpia, profesional y, sobre todo, orientada a la acción rápida que requiere un servicio de emergencias.

## Contexto y Objetivo

Desarrollado como un proyecto de iniciativa personal para aplicar conocimientos avanzados de Diseño Web y Experiencia de Usuario (UX/UI) sobre un caso de estudio real. 

El reto consistió en analizar las deficiencias de la web original (navegación compleja, falta de adaptación a dispositivos móviles o diseño anticuado) y proponer una solución técnica y visual que mejore la confianza del cliente y facilite el contacto inmediato.

## Características Destacadas (La Experiencia del Usuario)

* **Enfoque Mobile-First (Prioridad Móvil):** Dado que la mayoría de los usuarios que buscan una grúa lo hacen desde su teléfono en una situación de urgencia, la web está optimizada para cargar rápido y verse perfecta en pantallas pequeñas.
* **Llamadas a la Acción (CTA) Claras:** Botones de contacto y números de teléfono altamente visibles y accesibles para reducir el tiempo que tarda el usuario en pedir asistencia.
* **Modernización de la Identidad Visual:** Uso de una paleta de colores y una estructura que transmiten profesionalidad, seguridad y confianza, valores fundamentales en el sector del transporte de vehículos.
* **Arquitectura de la Información Mejorada:** Reestructuración de los servicios ofrecidos (asistencia, transporte, rescate) para que el visitante entienda la oferta de un solo vistazo.

## Tecnologías y Herramientas (Resumen Técnico)

Para el equipo técnico que revise el código, este rediseño se ha construido desde cero aplicando buenas prácticas de desarrollo front-end:

* **HTML5:** Estructura semántica adaptada a estándares modernos y SEO básico.
* **CSS3:** Maquetación, animaciones fluidas y diseño adaptable (Media Queries).
* **JavaScript:** Lógica de interactividad para menús y elementos dinámicos de la interfaz.
